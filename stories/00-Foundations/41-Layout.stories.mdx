# Layout Suggestions

The grid is flexible enough to accomodate various based layouts, to prioritize and emphasize certain pieces of content on a page. Multiple layouts can be used on one page. All Layouts should align with the Hard/Fixed Grid.  

  
## Examples of Layouts Currently In Use

### List Layout
These are typically a maximum of 12 columns wide on desktop, 8 columns on mobile. Great when creating a list for any type of content .
